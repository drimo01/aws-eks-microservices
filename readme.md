# Deploying Microservices on Amazon EKS

## Project Overview
This project demonstrates how to deploy a containerized application using Amazon Elastic Kubernetes Service (EKS).
The goal is to showcase modern, cloud-native deployment practices including containerization, orchestration, and scalability.

---

## Technologies Used
- AWS EKS
- Docker
- Kubernetes
- Helm
- Git & GitHub
